agi-bio

Version: 0-1.b5c6f3d

Synopsis: Genomic and proteomic data & pattern mining
Commit: b5c6f3d

Description

Genomic and proteomic data and pattern mining tools for AGI systems. This package provides specialized tools and algorithms for analyzing biological data, including genomic and proteomic information, using OpenCog's AGI framework for pattern discovery and analysis.
